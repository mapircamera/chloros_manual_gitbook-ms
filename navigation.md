Mengklik pada bar kemajuan atas akan membekukan panel dropdown; Mengklik lagi akan mengosongkannya.

## Menu sampingan

Menu bar sisi kiri mengandungi beberapa ikon untuk berinteraksi dengan:

#### <img src = ". gitbook/aset/icon_project-settings.jpg" alt = "" data-size = "line"> [Tetapan Projek] (Projek-Settings/Project-Settings.md)

Tab Tetapan Projek membolehkan anda menyesuaikan tetapan global dan pemprosesan untuk projek tersebut. Tetapkan sebelum anda mula memproses fail anda.

#### <img src = ". gitbook/aset/icon_file-browser.jpg" alt = "" data-size = "line"> file explorer

Tambah fail/folder dan keluarkan fail dari projek. Fail pendua diabaikan. Semak kotak di lajur sasaran untuk sebarang imej sasaran, dan pemprosesan hanya akan mencari imej yang ditandakan sebagai sasaran, sangat mempercepatkan masa pemprosesan.

#### <img src = ". gitbook/aset/icon_image-viewer.jpg" alt = "" data-size = "line"> [Image Viewer] (image-viewer-gui/pembukaan-image-full-screen.md)

Apabila anda mengklik pada imej dalam penonton imej utama, ia membuka skrin penuh dalam tab Image Viewer.

#### <img src = ". gitbook/aset/icon_log.jpg" alt = "" data-size = "line"> log debug

Semak log untuk cetakan debug apabila masalah berlaku. Salin/muat turun log dan hantar ke [Sokongan Teknikal Mapir] (https://www.mapir.camera/community/contact) untuk bantuan.

#### <img src = ". gitbook/aset/icon_user.jpg" alt = "" data-size = "line"> [pengguna login] (chloros+-login.md)

Sidebar log masuk pengguna membolehkan anda log masuk ke akaun Chloros+ anda untuk membuka kunci ciri lanjutan. Anda juga boleh melihat versi semasa aplikasi, serta menyesuaikan bahasa teks yang dipaparkan di Chloros GUI dan CLI.
